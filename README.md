# 抖音项目功能说明

## 📱 应用概览

本项目是一个简单版本抖音的前端网页，采用Modern.js框架 + React + Tailwind CSS构建。

**访问地址**: http://localhost:8080/

---

## 🎯 主要功能模块

### 1. 左侧导航菜单

左侧边栏包含9个导航选项，点击不同选项切换到不同的功能页面：

| 图标 | 名称 | 状态 | 说明 |
|------|------|------|------|
| ⭐ | **精选** | ✅ 已实现 | 网格阵列展示所有视频 |
| 🔥 | **推荐** | ✅ 已实现 | 横屏视频播放，支持键盘上下切换 |
| 🤖 | AI抖音 | 未实现 | 
| ❤️ | 关注 | 未实现 |  
| 👥 | 朋友 | 未实现 | 
| 👤 | 我的 | 未实现 |  
| 📹 | 直播 | 未实现 | 
| 🎬 | 放映厅 | 未实现 | 
| 📺 | 短剧 | 未实现 |  

**交互特性**：
- 当前选中的导航项会高亮显示（粉色背景 + 左侧粉色指示条）
- 悬停效果：缩放 + 背景色变化
- 平滑过渡动画

## ⭐ 精选页面

### 功能特点
- **网格布局**：响应式网格阵列，自动适配屏幕大小
  - 小屏：2列
  - 中屏：3-4列
  - 大屏：5-6列
- **视频卡片**包含：
  - 视频封面（9:16比例）
  - 视频标题（最多显示2行）
  - 作者头像和名称
  - 点赞数

### 交互功能
1. **悬停效果**：
   - 卡片放大（scale-105）
   - 封面图片放大（scale-110）
   - 显示黑色遮罩 + 粉色播放按钮
   - 粉色阴影效果

2. **点击播放**：
   - 全屏弹窗显示视频播放器
   - 右上角关闭按钮（✕）
   - 支持评论功能
   - ESC键或点击关闭按钮退出

### 视频播放器特性
- **控制按钮**：
  - 播放/暂停
  - 音量控制（悬停显示音量条）
  - 倍速选择（0.5×-2×）
  - 横竖屏切换（📱/🖥️）
  - 网页全屏（⤢）
  - 浏览器全屏（🔲）
- **进度控制**：
  - 点击跳转
  - 拖动调节
  - 显示缓冲进度
- **快捷键**：
  - 空格：播放/暂停
  - 滚轮：调节音量

---

## 🔥 推荐页面

### 功能特点
- **横屏播放**：16:9宽屏视频播放器
- **自动播放**：切换视频后自动播放
- **原生控制**：使用HTML5原生视频控制条

### 交互功能

#### 键盘控制
- **↑（上键）**：切换到上一个视频
- **↓（下键）**：切换到下一个视频
- **空格**：播放/暂停（原生控制）

#### 鼠标控制
1. **切换按钮**：
   - 底部中央显示 ↑ 和 ↓ 按钮
   - 显示当前进度（如：3 / 20）
   - 禁用状态：首个视频时上键禁用，末尾视频时下键禁用

2. **右侧操作栏**：
   - **点赞按钮**：
     - 默认：🤍白色心形
     - 点赞后：❤️粉色心形 + 缩放动画
     - 显示点赞数
   - **评论按钮**：
     - 💬图标
     - 显示评论数量
     - 点击打开右侧评论面板
   - **分享按钮**：
     - ↗️图标
     - UI展示（功能待实现）

### 视频信息展示
- **作者信息**：
  - 头像（圆形，48x48）
  - 作者名称（@username格式）
- **视频信息**：
  - 标题（大号加粗）
  - 描述（小字，最多显示2行）

---

## 💬 评论面板

### 功能特点
- **固定右侧**：宽度380px的固定面板
- **可切换显示**：点击评论按钮打开/关闭
- **自定义滚动条**：半透明样式

### 面板结构

#### 头部
- 显示"评论"标题和评论总数
- 显示当前视频标题
- 关闭按钮（✕）

#### 评论列表
每条评论包含：
- 用户头像（圆形，32x32）
- 用户名
- 评论内容（支持换行）
- 发布时间
- 操作按钮：
  - **点赞**：🤍/❤️图标，带数量显示和动画
  - **隐藏**：隐藏不想看的评论
  - **更多（⋯）**：复制评论功能

#### 底部输入框
- 圆角输入框
- 发布按钮（有内容时变为粉色并启用）
- 支持实时发布评论（显示为"游客"）

### 交互特性
1. **点赞评论**：
   - 点击后心形从🤍变为❤️
   - 点赞数+1/-1
   - 脉冲动画效果

2. **隐藏评论**：
   - 点击后显示"该评论已被隐藏"
   - 保留评论占位空间

3. **复制评论**：
   - 点击⋯按钮显示菜单
   - 选择"复制评论"
   - 底部显示"✓ 复制成功"提示（1.5秒后消失）

4. **发布评论**：
   - 新评论显示在列表顶部
   - 自动滚动到顶部
   - 清空输入框

## 🔍 搜索功能

### 位置
顶部导航栏中央

### 功能特点
1. **搜索输入框**：
   - 左侧🔍图标
   - 占位文本："搜索视频、用户或话题"
   - 聚焦时粉色边框 + 阴影效果

2. **搜索历史**：
   - 点击输入框自动显示
   - 最多保存10条历史记录
   - 显示序号（1.、2.、3...）
   - 显示记录总数
   - 淡入动画效果（fadeIn）

3. **历史记录操作**：
   - 点击任意历史记录快速搜索
   - 新搜索自动去重并置顶
   - 悬停时背景变色 + 箭头图标变粉色

### 层级管理
- z-index: 100（输入框）
- z-index: 110（下拉历史）
- 确保在所有元素之上

## 🎨 视觉设计

### 配色方案
- **主色调**：黑色背景（#000000）
- **强调色**：粉色（Pink-500）
- **辅助色**：白色半透明（rgba(255,255,255,0.1-0.8)）
- **渐变背景**：黑色到灰色渐变

### 动画效果
1. **淡入动画**（fadeIn）：
   - 从透明到不透明
   - 从上到下位移
   - 持续时间：0.3秒

2. **心形动画**（heart-pop）：
   - 点赞时缩放效果
   - 持续时间：0.25秒

3. **弹跳动画**（bounce）：
   - 占位页面图标
   - 无限循环
   - 持续时间：2秒

4. **脉冲动画**（pulse）：
   - 加载指示器
   - 透明度变化
   - 无限循环

5. **悬停效果**：
   - 缩放（scale-105/110）
   - 背景色变化
   - 阴影增强
   - 持续时间：150ms

### 响应式设计
- **断点**：
  - sm: 640px
  - md: 768px
  - lg: 1024px
  - xl: 1280px
- **自适应布局**：
  - 精选页面网格列数自动调整
  - 视频播放器尺寸自适应
  - 字体大小响应式

## ⌨️ 快捷键说明

| 快捷键 | 功能 | 适用页面 |
|--------|------|----------|
| **空格** | 播放/暂停 | 精选、推荐 |
